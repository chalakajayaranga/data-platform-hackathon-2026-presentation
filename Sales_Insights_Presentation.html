The Challenge

Transform how Reuters analysts work with RUM

Build an intelligent semantic layer on Snowflake that removes traditional query barriers. Create a natural-language interface that lets analysts explore visual content performance, trends, and user engagement patterns through conversational questions without needing complex SQL.

Goal

Reuters Unified Marketplace (RUM) enhanced data ecosystem accessible to every analyst.

Experience

Ask in plain language; get visual content performance, trends and engagement insights instantly while keeping your chat history

Approach 2

Why Streamlit Approach?

Streamlit Benefits

  • Full control on both frontend and backend logic
  • Showing visual diagrams inside the chat UI seamlessly
  • Ability to export queried data as CSV
  • Ability to enhance semantic views for any new schemas with relationships
  • Maintain chat history
  • Navigate between chats & continue

By using Plotly

  • No need for moving to external visualization tools like Power BI
  • No more expert knowledge
  • No extra cost

Outcomes

Business Impact

  • Faster time-to-insight - analysts get answers in seconds via natural language
  • Democratized analytics - no SQL expertise required; every analyst can query RUM data
  • Lower cost - no external BI tools or specialist dependency
  • Better decisions - real-time visual content performance, trends & engagement insights
  • Self-maintaining semantic layer - easy to extend for new schemas & relationships
  • Scalable adoption - more analysts can leverage RUM's enhanced data ecosystem
11 / 14
Roadmap

Future Works

  • Add dynamic tables with indexes to improve query performance
  • Call OpenArena chains from Streamlit instead of Cortex Complete to integrate with customized LLMs for different use cases
  • Integrate with Power BI to visualize data
  • Persist chat history in Snowflake database to maintain conversation context across sessions
12 / 14
Learnings

Challenges Faced

  • Team had only Power BI free licenses - couldn't integrate the Streamlit app with Power BI
  • Outbound traffic is blocked on Snowflake warehouse - couldn't integrate with OpenArena chains
13 / 14
Learnings

Lessons Learned

  • A clear semantic view is essential for reliable NLP-to-SQL conversion
  • Streamlit gave us full control over chat UX and visualization, unlike the AI Agent
  • Licensing and network restrictions shaped our integration options more than we expected
  • Team spirit mattered: parallel work, mutual support, and creative problem-solving when we hit blockers
